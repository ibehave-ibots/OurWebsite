# iBOTS Website Builder


## Installation

```
pip install .
```

## Build Site once

```
python main.py --cmd render
```

## Rebuild site while working on it, and serve it

```
python main.py  
```

or 

```
python main.py --cmd serve 
```


## For Developers

## Install with all dev dependencies, in editable mode
```
pip install -e .[dev]
```

## Run tests

```
pytest
```
